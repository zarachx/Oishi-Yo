# Oishi-Yo
A simple apps to find a restaurant


### 1. Clone the project
```
$ git clone https://github.com/zarachx/Oishi-Yo.git
```
or fork this repository and submit PR

### 2. Installation
on terminal, type command below
```
$ npm install
```

### 3. Development
```
$ npm run start-dev
```
then open app on http://localhost:8081/index.html

### 4. Production
first build the app
```
$ npm run build
```
running build app
```
$ npm run start
```

then open app on http://localhost:8081/index.html

##### Enjoy.
